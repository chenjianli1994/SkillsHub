# 需求规格书 Word 模板

> 本文件是 Word 直出模板的结构说明。实际 `.docx` 输出必须由 `scripts/按Word模板生成规格书.py` 读取结构化 JSON 后直接生成，不从 Markdown 转换。

## 排版规范

| 元素 | 规范 |
| --- | --- |
| 页面 | A4；左右边距约 2cm；上下边距约 2.25cm |
| 正文字体 | Microsoft YaHei / Aptos；10.5pt；1.3 倍左右行距 |
| 一级标题 | 15pt；蓝色；加粗；段前留白；下边框强调 |
| 二级标题 | 13pt；蓝色；加粗；与正文保持清晰间距 |
| 三级标题 | 11.5pt；加粗；用于功能需求编号等条目 |
| 列表 | 使用 Word 编号/项目符号；左缩进 0.5 英寸；悬挂缩进 |
| 表格 | 100% 页面宽度；固定列宽；浅蓝表头；浅灰首列；单元格内边距 |
| 页眉 | 文档名、版本、状态 |
| 页脚 | `第 X / Y 页` 页码 |
| 目录 | 使用 Word TOC 域；目录标题蓝色；目录项使用点线引导符和右对齐页码 |
